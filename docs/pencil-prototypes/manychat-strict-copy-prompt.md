# Strict Manychat Layout Copy Prompt for Pencil

## 任务

生成一个高保真 SaaS 首页原型，必须严格参考 Manychat 首页的版式骨架和商业转化节奏。

重点：这不是“重新设计一个 AI 目标导师官网”。这是“把 Manychat 的页面骨架换成 Helo Mentor 的内容和品牌色”。

不要自由发挥新的布局。不要做普通 SaaS dashboard hero。不要做卡片堆砌。不要做之前那种重新设计。

## 版权边界

可以复刻：

- 页面模块顺序。
- 布局比例。
- Hero 视觉骨架。
- 大圆角色块。
- 浮动聊天气泡的表达方式。
- CTA 节奏。
- 证言卡、before/after 对比、模板卡、FAQ 的结构。

不能复制：

- Manychat logo。
- Manychat 品牌名。
- Manychat 原文文案。
- Manychat 客户头像、真实客户、品牌 logo、案例数据。
- Manychat 产品截图。
- Manychat 紫色品牌体系。

## 画布

- Desktop frame：1440px 宽。
- 页面高度：4200px 到 5200px。
- 背景：#F7F4EC，接近温暖米白，不要纯白。
- 最大内容宽度：1180px。
- 页面整体要像成熟转化型 SaaS 官网。

## 品牌替换

产品名：Helo Mentor

品牌色：

- Primary green：#4A8C7C
- Dark green：#243634
- Soft green：#E8F3F0
- Cream：#F7F4EC
- Accent yellow：#E8B87A
- Text black：#171717
- Muted text：#5F6F6B

字体感觉：

- 标题使用厚重、圆润、现代的无衬线。
- 中文标题要大、黑、紧凑。
- 正文清晰，不要太细。

## 页面模块顺序，必须按这个来

### 1. 顶部导航

Manychat 式导航骨架：

- 页面顶部，白色/米白背景。
- 左侧：Helo Mentor logo，logo 可以是一个小写 h 的抽象标记，不要复杂。
- 中间：Product、Solutions、Resources、Pricing 四个导航项，带小下拉箭头感觉。
- 右侧：Log in 文本按钮 + Get started 绿色圆角按钮。
- 导航高度约 76px。
- 按钮圆角 999px。

注意：导航不要像后台产品，不要太细太冷。

### 2. Hero，大圆角绿色主视觉区

这是最重要的，必须像 Manychat 的首屏骨架：

- 在导航下方放一个巨大的绿色圆角 Hero 容器。
- 容器宽约 1320px，高约 760px，居中。
- 圆角非常大，约 36px 到 48px。
- 背景主色用 #4A8C7C 或更亮一点的绿色，不要渐变光球。
- 左侧是大标题和 CTA。
- 右侧是一个“导师人物/产品场景合成视觉”，不要做 dashboard 截图为主。

Hero 左侧：

- 大号标签：AI goal mentor for real progress
- 超大黑色/深绿标题，约 72px 到 92px：
  “把目标变成每天会发生的行动”
- 副标题，约 20px：
  “创建你的 AI 导师，输入一个短期目标。Helo 会拆计划、陪你执行、通过提问验收，并在你停下来时温和提醒。”
- CTA 组：
  主按钮：免费开始
  次按钮：查看如何运作
- CTA 按钮必须很大，像 Manychat，圆角 pill，高 56px。

Hero 右侧视觉必须复刻 Manychat 的“真人/创作者 + 聊天气泡 + 平台气泡”的感觉，但内容换成我们自己的：

- 中间放一个圆形或圆角人物卡：AI 导师“林澈”，可以用自制插画/头像色块，不要用真实人物照片。
- 周围漂浮 5 个聊天/任务气泡：
  - “今天只做 25 分钟”
  - “第 8 / 30 天”
  - “验收通过”
  - “08:00 已发送今日任务”
  - “明天任务已调整”
- 气泡要有阴影，白底，圆角 20px。
- 右下方可以放一个手机聊天卡片，显示导师消息和用户回复，像社媒聊天自动化视觉，但内容是目标陪跑。

Hero 底部：

- 做一条弧形/波浪/米白色切口，让绿色 Hero 和下方内容衔接，接近 Manychat 的活泼感。

### 3. 社交证明 / 信任条

参考 Manychat 的“客户/创作者证明”区域，但不要假造品牌 logo。

布局：

- 米白背景。
- 居中标题：
  “为那些总是开始、却很难坚持的人设计”
- 下方 4 个小 pill：
  英语学习、面试冲刺、习惯养成、健身入门。

### 4. Creator/Testimonial 风格卡片区

Manychat 常用创作者/案例卡片制造信任。我们用“用户状态卡”替代。

布局：

- 三列大卡片。
- 每张卡像人物案例，但不要真实照片，用抽象头像和状态。
- 卡片要有暖色背景、大圆角、强视觉。

卡片内容：

- “30 天英语口语”：从不敢开口，到每天复述 5 句。
- “15 天面试准备”：每天一轮模拟问答，减少临场慌乱。
- “21 天早睡计划”：把提醒、复盘和压力来源记录在同一个地方。

每张卡底部有一个小进度条或完成徽章。

### 5. Before / After 对比区

Manychat 的转化页常强调使用前后变化。这里必须做一个左右对比大区块。

左侧：

- 标题：Before
- 内容：目标写在备忘录里，几天后忘记；AI 聊天很热闹，但没有验收；打卡只是点一下。
- 背景：浅灰/米白，较弱。

右侧：

- 标题：After Helo
- 内容：每天知道下一步；导师记住你的节奏；完成后必须回答问题验收；停下来会收到提醒。
- 背景：品牌浅绿 #E8F3F0。

中间可以有箭头或分割线。

### 6. How it works 三步区

必须像 Manychat 的“简单启动流程”，不要做四张普通卡片。

标题：

“三步开始你的第一个目标”

三步：

1. Pick your mentor / 选择导师
2. Set your goal / 设定周期目标
3. Show up daily / 每天执行并验收

每一步用大号数字 01、02、03，配一个小产品界面片段。

### 7. Templates / 场景模板区

参考 Manychat templates/solutions 的商业结构。

标题：

“从一个模板开始，不用从空白开始”

布局：

- 2 行 3 列模板卡。
- 每张卡有类别标签、目标标题、周期、导师风格、CTA。

模板：

- 30 天英语口语
- 15 天面试准备
- 7 天作品集冲刺
- 21 天早睡计划
- 100 天健身入门
- 自定义目标

### 8. FAQ 区

参考 Manychat FAQ 的转化兜底。

标题：常见问题

5 个折叠项样式，默认展开第一项：

- 它和普通 ChatGPT 聊天有什么区别？
- AI 怎么判断我真的完成了？
- 可以创建多个导师吗？
- 邮件提醒会不会打扰？
- 第一版免费吗？

### 9. 底部大 CTA

必须像 Manychat 的最后转化区：

- 大圆角绿色容器。
- 大标题：
  “今天开始，让目标真的往前走一步”
- 副标题：
  “先免费创建一位导师，体验一次从计划到验收的完整闭环。”
- 大 pill 按钮：免费开始
- 旁边一个小文本链接：查看定价

### 10. Footer

简单 footer：

- Helo Mentor
- Product、Pricing、Privacy、Terms
- 一行说明：AI 目标导师不替代医疗、心理或法律等专业建议。

## 视觉验收标准

生成结果必须满足：

- 第一眼能看出是参考 Manychat 的大色块、人物/聊天气泡、转化页结构。
- Hero 不能是普通 dashboard 截图。
- 页面要活泼、有商业转化感，但颜色换成 Helo 的绿色/米白。
- 模块顺序必须完整。
- 不要出现 Manychat 字样。
- 不要出现紫色主题。
- 不要出现真实客户 logo 或虚假数据。
