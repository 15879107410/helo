# Study Notes

这里存放 Helo 首页从 0 到 1 的学习记录、阶段总结和问题复盘。

## 约定

- 每次你要求“总结一下”“记录一下”“写成文档”，我就把内容整理成 `md` 放进这个文件夹。
- 文件名建议使用日期前缀，方便按时间回看。
- 结构建议固定，便于学习和检索。

## 推荐命名

`YYYY-MM-DD-主题.md`

例如：

- `2026-05-07-homepage-refactor.md`
- `2026-05-07-third-screen-fix.md`

## 推荐内容结构

1. 做了什么
2. 为什么这么做
3. 遇到什么问题
4. 怎么解决的
5. 关键代码或文件
6. 验证结果
7. 下一步

## 使用方式

- 如果你只说“帮我总结今天做的事”，我就直接生成一篇记录。
- 如果你说“写到 study 里”，我就按这个目录落文件。
- 如果你想复盘某一屏，我会尽量单独拆成一篇。
